# Подсказки по командной сроке

команда смены директории
```sh
cd c:/folder_name
```

команда отобрадения текущей директории. MacOS, Linux
```sh
pwd
```

Листинг текущей директории
windows:
```sh
dir
```
linux, MacOS:
```sh
ls
```

удаление файла в windows:
```sh
del <filename>
```
в Linux, MacOS:
```sh
rm <file_name>
```
